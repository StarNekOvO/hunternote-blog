# CTFshow

[CTFshow](https://ctf.show/challenges) 是一个专注于 CTF 竞赛的在线练习平台，提供丰富的 CTF 题目和实战环境，适合各个水平的 CTF 爱好者练习。

## 平台特色

- **私教课程**：提供详细的 Web 和 Pwn 方向私教课程，适合系统化学习
- **题目丰富**：涵盖 Web、Crypto、Reverse、Pwn、Misc 等多个方向
- **实战环境**：提供真实的实战环境，帮助提升实战能力
- **适合各水平**：从入门到高级，适合不同水平的 CTF 爱好者
