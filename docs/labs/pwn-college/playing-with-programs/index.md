# Playing With Programs

## 模块概述

Playing With Programs 模块分为两个主要子模块：程序误用（Program Misuse）和程序交互（Program Interaction），旨在提升对软件交互的理解和创新性问题解决能力。

## 程序误用 (Program Misuse)

这个子模块包含一系列挑战，其中标准程序被设置为 set-user-ID (suid) 权限。目标是利用这些程序的提升权限来读取 flag。

### 典型挑战程序：
- **cat**：直接读取 flag 文件
- **more**：利用分页查看器读取 flag
- **wc**：字数统计工具的权限滥用
- **gcc**：编译器的权限利用
- **as**：汇编器的权限滥用
- **wget**：下载工具的利用
- **ssh-keygen**：通过加载自定义插件代码访问 flag

建议通过查阅每个程序的手册页（`man` pages）来理解其功能和潜在漏洞。

## 程序交互 (Program Interaction)

这个子模块专注于理解程序间的相互作用以及与系统的交互。挑战按级别划分，复杂度逐渐增加。

### 挑战结构：
- **级别 1-32**：涉及输入/输出流的操纵、文件描述符的理解，以及各种命令行工具的使用

通过参与这些挑战，可以深入理解程序间通信和数据处理机制。

## 学习目标

完成 Playing With Programs 模块将帮助你：
- 深入理解程序功能和潜在安全漏洞
- 掌握程序间通信的基本原理
- 为更高级的网络安全主题打下基础
