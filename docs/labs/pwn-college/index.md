# pwn.college

[pwn.college](https://pwn.college) 是亚利桑那州立大学（ASU）提供的系统安全学习平台，专注于二进制安全、漏洞利用和系统安全研究，提供从基础到高级的实践课程。

## 平台特色

- **系统化学习**：提供系统化的二进制安全学习路径，从基础到高级循序渐进
- **实践导向**：强调"Practice makes perfect!"，通过大量实践掌握技能
- **内容全面**：涵盖程序滥用、shellcode、二进制逆向、内存破坏、漏洞利用等主题
- **全球参与**：已有来自 118 个国家的 4000+ 用户参与，社区活跃
- **成就系统**：提供带位系统（belt system），完成挑战可获得相应带位认证
- **内核安全**：提供内核安全相关挑战，适合深入学习系统安全

这里将记录 pwn.college 的学习心得。
