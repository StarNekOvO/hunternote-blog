# Linux Luminarium 🐧

17 个模块，128 个挑战

## 模块概述

Linux Luminarium 是 pwn.college 平台上的一个综合性模块，旨在深入理解 Linux 系统及其安全机制。通过实践挑战，帮助学习者掌握 Linux 系统的各个方面。

## 主要内容

### 进程管理
- 理解进程的创建、管理和终止机制
- 学习 Linux 进程模型和生命周期

### 文件系统
- 探索 Linux 文件系统的结构和权限机制
- 理解文件权限、所有权和访问控制

### 网络配置
- 学习 Linux 网络配置和协议
- 掌握网络服务的基本配置和管理

### 安全机制
- 分析 Linux 安全特性
- 理解用户权限、能力机制和沙箱技术

## 挑战特点

每个挑战都提供动手实践的机会，将理论知识应用于实际场景。通过完成这些挑战，可以提升 Linux 系统管理和安全技能，为后续的二进制安全学习打下坚实基础。
