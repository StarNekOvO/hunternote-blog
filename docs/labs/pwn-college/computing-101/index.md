# Computing 101 💻

7 个模块，69 个挑战

## 模块概述

Computing 101 是 pwn.college 平台的入门模块，专注于 x86_64 汇编语言和系统交互的基础知识。通过一系列精心设计的挑战，帮助学习者建立底层编程和系统调用的基础理解。

## 主要内容

### 汇编基础
- **第一个寄存器**：学习使用 `mov` 指令将数据移动到 `rax` 寄存器
- **寄存器间移动**：掌握在不同寄存器间传输数据（如 `rsi` 到 `rdi`）

### 系统调用
- **第一个系统调用**：通过设置 `rax` 中的系统调用号并执行 `syscall` 指令来调用 `exit`
- **退出码设置**：在调用 `exit` 前通过 `rdi` 寄存器设置退出码
- **链式系统调用**：顺序执行多个系统调用，如先输出再退出

### 内存操作
- **解引用寄存器**：通过包含内存地址的寄存器访问内存中的数据
- **字符串输出**：设置适当的寄存器并调用 `write` 系统调用来输出字符串

### 高级技巧
- **构建可执行文件**：使用 `as` 和 `ld` 等工具将汇编代码组装并链接为二进制可执行文件
- **高效取模**：使用位运算（特别是当除数是 2 的幂时）来计算模运算

## 挑战特点

这些挑战从基础逐步深入，构建对低级编程和系统交互的全面理解。通过动手编写汇编代码和执行系统调用，为后续的二进制利用和逆向工程学习奠定基础。
